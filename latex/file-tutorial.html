How to use a style and class file?

I will assume you have \(\LaTeX\) installed on your personal computer (or you are using an inferior option like Overleaf.)
  1. Firstly, navigate to the directory of your LaTeX file. For me I will use Windows; say it is in C:\Users\User\Documents\Project .
  2. Download either both, or one of the files. I will say I downloaded both.
  3. Go to the folder/directory that you downloaded it in and copy/cut/move the file(s) to your desired location; ours is C:\Users\User\Documents\Project.
  4. Go to your .tex file and insert in the preamble (if you are compiling both files):

    \documentclass{notescls}
    \usepackage{notesty}

    | If you are using only the .sty file:

    \documentclass{article}
    \usepackage{notesty}

    I'm using the article class as an example.

    | If you are using only the .cls file:

    \documentclass{notescls}
    \usepackage{amsthm}

    I'm using amsthm as an example.
  5. Compile your document using LuaLaTeX. If I remember poorly, XeLaTeX also works with some caveats (it is not recommended as it is old) and I have not tried LatexMK.
